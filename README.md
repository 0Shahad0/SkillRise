# SkillRise

SkillRise is an AI-powered learning platform designed to improve technical training by helping supervisors create, assign, and evaluate training tasks while monitoring trainee performance through intelligent analytics.

---

## Overview

The platform combines artificial intelligence with training management to simplify task creation, automate evaluations, and provide actionable insights into trainee progress.

---

## Key Features

- AI-powered task generation using Large Language Models (LLMs)
- Manual task creation and management
- Automated AI evaluation
- Trainee performance tracking
- Interactive analytics dashboard
- Skill-focused progress monitoring
